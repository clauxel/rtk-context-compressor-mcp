# Workflow

RTK Context Compressor is a hosted remote MCP for OpenAI Codex context compressor.

## Repeatable Flow

1. Open RTK Context Compressor and select the buyer plan.
2. Create or request a bearer token from the hosted product.
3. Add https://rtkcontext.clauxel.com/mcp to a compatible MCP client.
4. Run tools/list, then call compress_context_for_task with public-safe sample data.
5. Save the returned receipt or export for human review.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
